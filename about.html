About Project

A complete Web Engineering collaboration platform.

CollabChat is designed as a final semester project with authentication, live chat, workspaces, tasks, files, notifications, search, profile settings, and admin management.

Project Goal

The goal is to demonstrate modern web engineering concepts: responsive UI, modular JavaScript, backend-as-a-service integration, database design, authorization rules, real-time subscriptions, storage, and admin workflows.

Frontend

HTML5, CSS3, JavaScript modules, Vite, responsive dashboard, modals, cards, tables, and mobile layout.

Backend

Supabase Auth, Postgres database, Row Level Security, Realtime subscriptions, and Storage buckets.

Submission Ready

Includes SQL schema, setup instructions, all pages, and README documentation.
